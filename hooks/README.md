# Hooks

Custom React hooks for KidGuard.

## Planned Hooks

- **useAuth.ts** - Authentication state and methods
- **useChildMode.ts** - Child mode management
- **useExercise.ts** - Exercise generation and validation
- **useSession.ts** - Session management
- **useTimeBank.ts** - Screen time tracking

## Guidelines

- Follow React hooks naming convention (use*)
- Keep hooks focused on single responsibility
- Use TypeScript for type safety
- Document hook parameters and return values
- Test hooks with React Native Testing Library
