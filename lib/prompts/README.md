# DeepSeek Prompt Templates

This directory contains prompt templates for DeepSeek AI integration.

## Planned Templates

- **exercise-generation.ts** - Templates for generating exercises by type and difficulty
- **answer-validation.ts** - Templates for validating child answers with leniency
- **math-exercises.ts** - Math-specific exercise prompts
- **reading-exercises.ts** - Reading comprehension prompts
- **logic-exercises.ts** - Logic puzzle prompts
- **vocabulary-exercises.ts** - Vocabulary and word game prompts

## Guidelines

- All prompts should specify French language output
- Include age range and difficulty level parameters
- Define clear output format (JSON schema)
- Version prompts for A/B testing
- Document expected response formats
